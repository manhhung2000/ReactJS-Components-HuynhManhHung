1/ cài đặt project:
    npx create-react-app ten-app
2/ chạy project:
    2.1/ mở terminal ngay tại file package.json
    2.2/ gõ lệnh: npm run start
3/ tài liệu về reactjs: https://reactjs.org/docs/getting-started.html
4/ flow code reactjs: index.html => index.js => App.js
5/ reactjs sẽ không hiểu html, mà nó chỉ hiểu JSX ( class -> className )
6/ lý do dùng components là:
    - tái sử dụng
    - dễ quản lý
7/ điều kiện để tạo ra 1 component:
    - tái sử dụng
    - dễ quản 
8/ có 2 dạng component:
    - class
    - function    

bonus: 
1/ một website thường có 2 dạng:
    - single page application (chuyển trang mà không reload) (phải tải html, css, js từ server -> client)
    - multiple page application (chuyển trang bị reaload) (chỉ tải data từ server -> client)
2/ về SEO thì multiple page application tốt hơn single page application
3/ reactjs (single page application) thì nó không SEO tốt nhưng NETJS (https://nextjs.org/) giúp reactjs SEO tốt hơn
